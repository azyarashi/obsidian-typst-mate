## タブストップへの移動

カーソル

## ペアから抜ける

選択範囲がないときのみ
(選択範囲がタブストップのプレースホルダー由来の場合を除く)

## 数式・コードブロックから抜ける

### カーソルが末尾にない場合、抜ける前に末尾に移動する

```md
$a^2 |+ b^2 = c^2$

オフ
$a^2 + b^2 = c^2$|

オン
$a^2 + b^2 = c^2|$
```

### 単一行のディスプレイ数式から抜けるときは改行しない

```md
$$a^2 |+ b^2 = c^2$$

オフ
$$a^2 + b^2 = c^2$$
|

オン
$$a^2 + b^2 = c^2$$|
```
